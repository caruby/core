caRuby API

Each caRuby application component, e.g. caRuby Tissue, presents a JRuby facade for the caBIG application domain API. The caRuby application component is built on caRuby Core, which encapsulates the caCORE data access API. The caTissue API is documented in the caruby-tissue gem. The caCORE API is documented in the caruby-core gem.
