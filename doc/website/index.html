Overview

caRuby simplifies interaction with caBIG® application services. caRuby presents a JRuby caBIG façade that supports the following uses:

  • Migration from legacy systems
  • Incremental update from source applications
  • Extract from a caBIG database
  • Utility administrative tasks
  • Workflow data transformations
  • Lightweight web services
  • Site-specific user interfaces

Getting Started

  1. Peruse the Features to see what caRuby offers.
  2. Follow the Quick Start instructions to install caRuby.
  3. Consult the FAQ for usage tips.
  4. Visit caRuby Support if you run into a problem.
  5. Contribute a solution if you're feeling ambitious!
