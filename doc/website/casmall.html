caSmall Web Service

caSmall is a micro-framework for accessing and building lightweight caBIG application web services and applications. This initiative is in the planning stage. The initial offering will be a RESTful web service for reading and writing caTissue data. The web service resides on the caTissue application server. The client calls caRuby by sending a standard JSON request without installing caTissue or caRuby on the client workstation.

caSmall leverages the advent of ultra-slim Ruby frameworks such as Sinatra, HttParty and Thin. Stay tuned for further developments.
